# Manuel git

## Procédure

- Créer une branch
  - Par exemple :
  - fix/cors, feat/filters, ...
- Faire un merge dans develop :
  - être dans develop
  - git merge feat/filters
- Ou faire une pull request dans main :
  - depuis github, appuyer sur le bouton Contribute
  - Faire une pull request avec au moins 1 reviewer

## A ne pas faire

- Push directement dans main ou dans develop
- Modifier une branche dont vous n'êtes pas auteur sans rien notifier
